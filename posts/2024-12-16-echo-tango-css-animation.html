Basic CSS Animations

Posted on December 16, 2024
CSS Best Practices

How-To

  • Draw a circle in CSS
  • Define and use CSS variables
  • CSS animation using keyframes
  • Using background position for CSS animations
  • CSS radial gradient

HTML

<div class="loader"></div>

CSS

:root {
    /* Loader colors */
    --loader-color-1: #D1BFAD;
    --loader-color-2: #C696FF;
    --loader-color-3: #F5F0EC;
    --loader-color-4: #6EC86E;
    --loader-bg: #000000;
}

.loader {
    width: 40px;
    aspect-ratio: 1;
    border-radius: 50%;
    background:
      radial-gradient(farthest-side, var(--loader-color-1) 94%, #0000),
      radial-gradient(farthest-side, var(--loader-color-2) 94%, #0000),
      radial-gradient(farthest-side, var(--loader-color-3) 94%, #0000),
      radial-gradient(farthest-side, var(--loader-color-4) 94%, #0000),
      var(--loader-bg);
    background-size: 105% 105%;
    background-repeat: no-repeat;
    animation: l5 2s infinite; 
}

@keyframes l5 {
    0%  {background-position: 50% -50px,-40px 50%, 50% calc(100% + 50px),calc(100% + 50px) 50%}
    20%,
    25% {background-position: 50% -50px,-50px 50%, 50% calc(100% + 50px),50% 50%}
    45%,
    50% {background-position: 50% -50px,-50px 50%, 50% 50%              ,50% 50%}
    75%,
    75% {background-position: 50% -50px, 50%  50%, 50% 50%              ,50% 50%}
    95%,
    100%{background-position: 50%  50% , 50%  50%, 50% 50%              ,50% 50%}
}

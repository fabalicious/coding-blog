Modern CSS Features You Should Know

Posted on December 13, 2024
CSS Best Practices

CSS has evolved significantly in recent years. Here are some modern features that can enhance your web development workflow.

CSS Custom Properties (Variables)

:root {
    --primary-color: #4a90e2;
    --spacing-unit: 1rem;
}

.button {
    background-color: var(--primary-color);
    padding: var(--spacing-unit);
    border-radius: calc(var(--spacing-unit) / 2);
}

CSS Grid Layout

.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 1rem;
}

.grid-item {
    background: #f0f0f0;
    padding: 1rem;
    border-radius: 4px;
}
